# Integer-to-Roman
Convert any given integer into a roman numeral

<pre><p>
  Integer to Roman <br>
  1 - I <br> 
  5 - V <br>  
  10 - X <br>  
  50 - L <br> 
  100 - C <br> 
  500 - D <br> 
  1000 - M <br> 
</p></pre>

<pre><p>
  example1: 1012 = 
               = 1000(M) + 10(X) + 2(II) 
               = MXII
               
  example2: 910 = 900 + 10
             = [1000 - 100] + 10
             = [1000(M) - 100(C)] + 10(x)
             = CMX
             
 Note: In case of numbers like 4 and 9, we subtract the left character value from the right character value.
</pre></p>
